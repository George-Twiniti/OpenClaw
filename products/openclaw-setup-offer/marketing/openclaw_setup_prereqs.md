# OpenClaw Setup — Prerequisites (Client)

## Minimal requirements
**Windows**
- WSL2 installed (Ubuntu distro preferred)

**Mac**
- Homebrew installed

**Access**
- Admin permissions for installs

**Email (if you want email automation)**
- Gmail: App Password ready
- Exchange/O365: App registration with Client ID + Tenant ID + Client Secret

## Optional (if requested)
**Telegram channel**
- Create a bot via BotFather
- Provide the bot token
- Add the bot to your chat

**Slack (if requested)**
- Admin access to workspace
- Slack app token or OAuth setup

---
If you don’t know how to do any of this, we can guide you during the call — just make sure **WSL2 (Windows)** or **Homebrew (Mac)** is in place first.
