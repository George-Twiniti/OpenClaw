# OpenClaw Setup — Delivery Checklist

## Pre‑call
- [ ] Confirm OS + hardware
- [ ] Confirm access method (screen share / SSH)
- [ ] Confirm prerequisites completed (see prereq doc)
- [ ] Collect required credentials
- [ ] Confirm desired channels (Telegram/Webchat/Slack)

## Install
- [ ] Install OpenClaw + gateway
- [ ] Verify `openclaw status`
- [ ] Configure base models + fallbacks

## Workspace
- [ ] Populate AGENTS.md, SOUL.md, IDENTITY.md, USER.md
- [ ] Ensure MEMORY.md exists
- [ ] Set up daily memory logs

## Automation
- [ ] Startup brief scheduled
- [ ] Daily wrap‑up scheduled
- [ ] Heartbeat configured

## Security
- [ ] Secrets stored safely (password manager or SOPS+AGE)
- [ ] Minimal permissions set

## Validation
- [ ] Test a sample workflow (startup brief)
- [ ] Test a memory write + recall
- [ ] Confirm cron delivery

## Handoff
- [ ] Walkthrough of key files + flows
- [ ] Maintenance checklist shared
