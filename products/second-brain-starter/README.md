# Second‑Brain Starter (Solo Founder Edition)

**Promise:** Capture chaos → daily clarity.

## What’s inside
- **Folder structure** for a Second‑Brain repo
- **Capture templates** (AI convo, meeting, idea, research)
- **Daily synthesis + handoff** templates
- **Automation scripts** (capture, synthesis stub, commit/push helper)
- **Quick‑start guide** (15‑minute install)

## Quick start (local)
1) Copy the `templates/` into your Second‑Brain repo
2) Review `docs/quickstart.md`
3) Run the starter scripts in `scripts/`
