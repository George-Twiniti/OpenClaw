# Second‑Brain Starter — Quick Start (15 minutes)

## 1) Create your repo
- Create a private GitHub repo named `second-brain`
- Clone it locally

## 2) Copy the starter structure
Copy the included `templates/` folder into your repo root.

## 3) Install scripts
Copy `scripts/` into your repo root.

## 4) Daily flow (recommended)
- **Capture** key inputs during the day using the templates
- **Synthesize** at end of day
- **Handoff** for tomorrow

## 5) Automation (optional)
Use the provided scripts for:
- Creating new capture files
- Generating daily synthesis stubs
- Committing + pushing

## 6) Suggested cadence
- Morning: glance at yesterday’s handoff
- End of day: write synthesis + handoff
