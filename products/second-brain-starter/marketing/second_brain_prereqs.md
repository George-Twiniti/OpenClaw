# Second‑Brain Starter — Prerequisites

## Core requirements
- **OpenClaw installed**
- **GitHub account** (for workflows + backups)
- **Python 3**

## Slack Lite (optional, no n8n)
To use Slack Lite (webhook capture):
- **Cloudflare Tunnel (cloudflared)** for a public webhook URL
- **Slack app** with a workflow or webhook integration

## Advanced Slack (n8n)
If you want the full Slack → Notion/SignalDesk routing:
- **n8n account** + credentials

If you’re not sure how to set these up, we can provide setup support.
