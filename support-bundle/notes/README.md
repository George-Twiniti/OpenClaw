# OpenClaw Support Project Bundle

## What this is
A local docs snapshot + troubleshooting prompt pack to reduce hallucinated fixes.

## How to use (Claude/Codex/any LLM)
1) Create a “Support” project.
2) Upload the `docs/` folder (or the zip).
3) Use prompts in `prompts/`.

## Suggested workflow
- Start with the **Triage** prompt.
- Ask for **root cause**, **minimal fix**, and **verification steps**.
- Always request a **single command** at a time for safety.
