# Firecrawl

Use Firecrawl when you need to extract web pages into structured text for downstream content work.

## Core workflow

1. Initialize or verify Firecrawl access.
2. Extract the target page or site.
3. Preserve the extracted source text.
4. Feed the result into the AEO or wiki workflow.

## Keep it minimal

- Use Firecrawl as an extraction layer, not as the final content system.
- Keep raw extracted pages separate from generated pages.
- Add more detailed automation only after the first real use case.
