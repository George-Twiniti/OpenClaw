## Video capture paths
- Transcript: `conv/video/YYYY-MM-DD/<slug>.md`
- Summary: `summaries/video/YYYY-MM-DD/<slug>.md`

## Summary requirements
- Creator-ready summary optimized for slide creation + LinkedIn/Substack posts.
- Include insights, implications, framing, standout quotes, and actionable takeaways.
- Always include a **Tags** line (see below).

## Tagging rules
Always include a **Tags** line in video summaries.
Core tags: `#ai` `#video` `#workflow` `#research` `#product` `#strategy` `#book` `#writing`
Add 1–3 topic-specific tags when obvious.

## Commit conventions
- Capture commits: `capture(ai): YYYY-MM-DD HH:mm`
- Synthesis commits: `synthesis(daily): YYYY-MM-DD`
